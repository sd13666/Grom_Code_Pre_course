Стилі елементу можна задатис за допомогою:

  • AАтрибута 'style'
  • Тега 'style'
  • Використовувати окремий файл з розширенням .css

CSS свойства "width" та "height" непрацюють для інлайнових елементів

Наслідування в CSS - механізм, з допомогою якого значення свойства елемента-батька передаються його елементам-дітям.
